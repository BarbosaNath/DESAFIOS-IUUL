Python

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. Ela foi criada com ênfase na legibilidade do código e na simplicidade, tornando-a uma das linguagens mais populares para desenvolvimento de software.

PySimpleGUI

PySimpleGUI é uma biblioteca Python que facilita a criação de interfaces gráficas de usuário (GUIs) de forma rápida e simples. Ela é projetada para ser fácil de aprender e usar, permitindo que desenvolvedores criem interfaces interativas para suas aplicações de maneira eficiente.
Visitar site

SQLite3

SQLite3 é uma biblioteca de software que implementa um sistema de gerenciamento de banco de dados relacional embutido em uma única biblioteca. Ela fornece uma maneira simples e eficiente de armazenar, gerenciar e acessar dados em um banco de dados local, sem a necessidade de um servidor de banco de dados separado.
Visitar site

Pillow

Pillow é uma biblioteca Python, baseada na biblioteca PIL (Python Imaging Library), amplamente utilizada para manipulação de imagens. Ela fornece uma ampla gama de recursos e funcionalidades para abrir, editar, processar e salvar imagens de diversos formatos.
Visitar site

Java

Java é uma linguagem de programação de propósito geral e orientada a objetos amplamente utilizada. Ela foi desenvolvida para ser portável, robusta e segura, tornando-a uma das linguagens mais populares para desenvolvimento de software. Uma das principais características do Java é sua máquina virtual Java (JVM), que permite que os programas escritos em Java sejam executados em diferentes plataformas sem a necessidade de recompilação.
Visitar site

AWT

AWT (Abstract Window Toolkit) é uma biblioteca de componentes gráficos para desenvolvimento de interfaces de usuário em Java. Ela fornece um conjunto de classes e métodos para criar janelas, botões, caixas de texto, menus e outros elementos de interface gráfica.
Visitar site

HTML

HTML (Hypertext Markup Language) é uma linguagem de marcação utilizada para estruturar e exibir o conteúdo de páginas da web. Ela é a base fundamental da internet e permite a criação de documentos estruturados por meio de tags, que definem o significado e a organização dos elementos presentes em uma página.
Visitar site

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a aparência visual e o layout de elementos em páginas da web. Ele permite que os desenvolvedores apliquem estilos como cores, fontes, tamanhos e posicionamentos aos elementos HTML, proporcionando uma experiência visual consistente e atraente aos usuários.
Visitar site

Bootstrap

Bootstrap é um framework de desenvolvimento front-end popular e de código aberto, utilizado para criar sites e aplicativos web responsivos e visualmente atraentes. Ele oferece um conjunto abrangente de componentes pré-projetados, como botões, menus, formulários e carrosséis, além de uma grade flexível que facilita a criação de layouts responsivos.
Visitar site
